Steps to execute the Index_usage Extractor

Step 1:Copy all raw files ,of the format [index_usage.txt.yyyyMMdd] to the folde Raw_files.

Step 2:Please note that there should be one file for each week,starting from the first date.

Step 3:Note down the first date[from the file name] and the number of files in the folder.

Step 4:Open Script folder ,and run Extractor.bat. Enter the required values when prompted.

Step 5:Find the Final output in the Output Folder,with name int format Results_Startdate-Enddate.xls
